fast_forward.index

Indexes

Indexes are the core of the Fast-Forward library. In the following, you'll find some snippets how to create and use indexes.

Index types

Currently, two types of indexes are available:

  • InMemoryIndex: Everything is held in memory entirely.
  • OnDiskIndex: Vectors are stored on disk and accessed on demand.

OnDiskIndexes can be loaded into memory using OnDiskIndex.to_memory.

Creating an index

The following snippet illustrates how to create an OnDiskIndex object (given a fast_forward.encoder.Encoder, my_query_encoder) and add some vector representations to it:

my_index = OnDiskIndex(Path("my_index.h5"), my_query_encoder)
my_index.add(
    my_vectors,  # shape (3, 768)
    doc_ids=["d1", "d1", "d2"],
    psg_ids=["d1_p1", "d1_p2", "d2_p1"]
)

Here, my_vectors is a Numpy array of shape (3, 768), 768 being the dimensionality of the vector representations. The first two vectors correspond to two passages of the document d1, the third vector corresponds to d2, which has only a single passage. It is also possible to provide either only document IDs or only passage IDs.

The index can then be subsequently loaded back using OnDiskIndex.load.

Using an index

An index can be used to compute semantic re-ranking scores by calling them directly. It requires a fast_forward.Ranking (typically, this comes from a sparse retriever) with the corresponding queries:

ranking = Ranking.from_file(Path("/path/to/sparse/run.tsv"), queries)
result = my_index(ranking)

Here, queries is a simple dictionary mapping query IDs to actual queries to be encoded. The resulting ranking, result, has the semantic scores for the query-document (or query-passage) pairs. Afterwards, retrieval and re-ranking scores may be combined (see Rankings).

If the input ranking has a large number of queries, one can use the batch_size parameter. The following example processes the ranking in batches of 16 queries each:

result = my_index(ranking, batch_size=16)

Ranking mode

Each index has a ranking Mode. The active mode determines the way scores are computed. For example, consider the example index from earlier. Setting the mode to Mode.PASSAGE instructs the index to compute scores on the passage level (and expect passage IDs in the input ranking):

my_index.mode = Mode.PASSAGE

Similarly, the index can return document IDs, where the score of a document computes as

  • the highest score of its passages (Mode.MAXP),
  • the score of the its first passage (Mode.FIRSTP) or
  • the average score of all its passages (Mode.AVEP).

Early stopping

Early stopping is a technique to limit the number of index look-ups. This can be beneficial for OnDiskIndexes, especially when the disk is slow. For early stopping, a relatively small cut-off depth (e.g., 10) is required, and it is mostly helpful when a large number of candidates are to be re-ranked. More information can be found in the paper. Note that the implementation here differs slightly from the algorithm in the paper, as the early stopping criterion is only computed at depths that are specified via the early_stopping_depths parameter for performance reasons.

The following snippet demonstrates early stopping with

  • a cut-off depth of 5,
  • interpolation parameter 0.2,
  • depths 400 and 5000.
result = my_index(
    ranking,
    early_stopping=5,
    early_stopping_alpha=0.2,
    early_stopping_depths=(400, 5000),
)

Specifically, in the first step, the top-400 documents/passages for each query are scored. Afterwards, the top-5000 documents/passages are scored only for those queries that do not meet the early stopping criterion yet.

279    def _compute_scores(
280        self, data: pd.DataFrame | pd.Series, query_vectors: np.ndarray
281    ) -> pd.DataFrame:
282        """Compute scores for a data frame.
283
284        The input data frame needs a "q_no" column with unique query numbers.
285
286        :param data: Input data frame (or series).
287        :param query_vectors: All query vectors indexed by "q_no".
288        :return: Data frame with computed scores.
289        """
290        # get all required vectors and corresponding IDs from the FF index
291        vectors, vec_ids = self._get_vectors(data["id"].unique())
292        if self.quantizer is not None:
293            vectors = self.quantizer.decode(vectors)
294
295        # merge data frames so "id_idx" can be used to index arrays
296        df_vec_ids = pd.DataFrame(vec_ids, columns=["id"])
297        df_vec_ids["id_idx"] = df_vec_ids.index
298        df_merged = df_vec_ids.merge(data[["id", "q_no"]], how="left", on="id")
299
300        # compute all dot products (scores)
301        q_reps = query_vectors[df_merged["q_no"].tolist()]
302        d_reps = vectors[df_merged["id_idx"].tolist()]
303        df_merged["ff_score"] = np.sum(q_reps * d_reps, axis=1)
304
305        # select aggregation operation based on current mode
306        if self.mode == Mode.MAXP:
307            op = "max"
308        elif self.mode == Mode.AVEP:
309            op = "mean"
310        else:
311            op = "first"
312        df_agg = df_merged.groupby(["id", "q_no"], as_index=False).aggregate(op)
313
314        return data.merge(df_agg, on=["id", "q_no"])

316    def _early_stopping(
317        self,
318        df: pd.DataFrame,
319        query_vectors: np.ndarray,
320        cutoff: int,
321        alpha: float,
322        depths: Iterable[int],
323    ) -> pd.DataFrame:
324        """Compute scores with early stopping for a data frame.
325
326        The input data frame needs a "q_no" column with unique query numbers.
327
328        :param df: Input data frame.
329        :param query_vectors: All query vectors indexed by "q_no".
330        :param cutoff: Cut-off depth for early stopping.
331        :param alpha: Interpolation parameter.
332        :param depths: Depths to compute scores at.
333        :return: Data frame with computed scores.
334        """
335        # data frame for computed scores
336        scores_so_far = pd.DataFrame()
337
338        # [a, b] is the interval for which the scores are computed in each step
339        a = 0
340        for b in sorted(depths):
341            if b < cutoff:
342                continue
343
344            # identify queries which do not meet the early stopping criterion
345            if a == 0:
346                # first iteration: take all queries
347                q_ids_left = pd.unique(df["q_id"])
348            else:
349                # subsequent iterations: compute ES criterion
350                q_ids_left = (
351                    scores_so_far.groupby("q_id")
352                    .filter(
353                        lambda g: g["int_score"].nlargest(cutoff).iat[-1]
354                        < alpha * g["score"].iat[-1] + (1 - alpha) * g["ff_score"].max()
355                    )["q_id"]
356                    .drop_duplicates()
357                    .to_list()
358                )
359            LOGGER.info("depth %s: %s queries left", b, len(q_ids_left))
360
361            # take the next chunk with b-a docs/passages for each query
362            chunk = (
363                df.loc[df["q_id"].isin(q_ids_left)]
364                .groupby("q_id")
365                .nth(list(range(a, b)))
366            )
367
368            # stop if no pairs are left
369            if len(chunk) == 0:
370                break
371
372            # compute scores for the chunk and merge
373            out = self._compute_scores(chunk, query_vectors)[["orig_index", "ff_score"]]
374            chunk_scores = chunk.merge(out, on="orig_index", suffixes=(None, "_"))
375
376            # compute interpolated scores
377            chunk_scores["int_score"] = (
378                alpha * chunk_scores["score"] + (1 - alpha) * chunk_scores["ff_score"]
379            )
380
381            scores_so_far = pd.concat(
382                [scores_so_far, chunk_scores],
383                axis=0,
384            )
385
386            a = b
387        return scores_so_far.join(df, on="orig_index", lsuffix="", rsuffix="_")

389    def __call__(
390        self,
391        ranking: Ranking,
392        early_stopping: int | None = None,
393        early_stopping_alpha: float | None = None,
394        early_stopping_depths: Iterable[int] | None = None,
395        batch_size: int | None = None,
396    ) -> Ranking:
397        """Compute scores for a ranking.
398
399        :param ranking: The ranking to compute scores for. Must have queries attached.
400        :param early_stopping: Perform early stopping at this cut-off depth.
401        :param early_stopping_alpha: Interpolation parameter for early stopping.
402        :param early_stopping_depths: Depths for early stopping.
403        :param batch_size: How many queries to process at once.
404        :raises ValueError: When the ranking has no queries attached.
405        :raises ValueError: When early stopping is enabled but arguments are missing.
406        :return: Ranking with the computed scores.
407        """
408        if not ranking.has_queries:
409            raise ValueError("Input ranking has no queries attached.")
410        if early_stopping is not None and (
411            early_stopping_alpha is None or early_stopping_depths is None
412        ):
413            raise ValueError("Early stopping requires alpha and depths.")
414        t0 = perf_counter()
415
416        # get all unique queries and query IDs and map to unique numbers (0 to m)
417        query_df = (
418            ranking._df[["q_id", "query"]].drop_duplicates().reset_index(drop=True)
419        )
420        query_df["q_no"] = query_df.index
421        df_with_q_no = ranking._df.merge(query_df, on="q_id", suffixes=(None, "_"))
422
423        # early stopping splits the data frame, hence we need to keep track of the
424        # original index
425        df_with_q_no["orig_index"] = df_with_q_no.index
426
427        # batch encode queries
428        query_vectors = self.encode_queries(list(query_df["query"]))
429
430        def _get_result(df: pd.DataFrame) -> pd.DataFrame:
431            if early_stopping is None:
432                return self._compute_scores(df, query_vectors)
433
434            assert early_stopping_alpha is not None
435            assert early_stopping_depths is not None
436            return self._early_stopping(
437                df,
438                query_vectors,
439                early_stopping,
440                early_stopping_alpha,
441                early_stopping_depths,
442            )
443
444        num_queries = len(query_df)
445        if batch_size is None or batch_size >= num_queries:
446            result = _get_result(df_with_q_no)
447        else:
448            # assign batch indices to query IDs
449            df_with_q_no["batch_idx"] = (
450                df_with_q_no.groupby("q_id").ngroup() / batch_size
451            ).astype(int)
452
453            chunks = []
454            num_batches = int(num_queries / batch_size) + 1
455            for batch_idx in tqdm(range(num_batches)):
456                batch = df_with_q_no[df_with_q_no["batch_idx"] == batch_idx]
457                chunks.append(_get_result(batch))
458            result = pd.concat(chunks)
459
460        result["score"] = result["ff_score"]
461        LOGGER.info("computed scores in %s seconds", perf_counter() - t0)
462        return Ranking(
463            result,
464            name="fast-forward",
465            dtype=ranking._df.dtypes["score"],
466            copy=False,
467            is_sorted=False,
468        )

174    def _add(
175        self,
176        vectors: np.ndarray,
177        doc_ids: IDSequence,
178        psg_ids: IDSequence,
179    ) -> None:
180        with h5py.File(self._index_file, "a") as fp:
181            # if this is the first call to _add, no datasets exist
182            if "vectors" not in fp:
183                self._create_ds(fp, vectors.shape[-1], vectors.dtype)
184
185            # check all IDs first before adding anything
186            doc_id_size = fp["doc_ids"].dtype.itemsize  # pyright: ignore[reportAttributeAccessIssue]
187            for doc_id in doc_ids:
188                if doc_id is not None and len(doc_id) > doc_id_size:
189                    raise RuntimeError(
190                        f"Document ID {doc_id} is longer than the maximum "
191                        f"({doc_id_size} characters)."
192                    )
193            psg_id_size = fp["psg_ids"].dtype.itemsize  # pyright: ignore[reportAttributeAccessIssue]
194            for psg_id in psg_ids:
195                if psg_id is not None and len(psg_id) > psg_id_size:
196                    raise RuntimeError(
197                        f"Passage ID {psg_id} is longer than the maximum "
198                        f"({psg_id_size} characters)."
199                    )
200
201            num_new_vecs = vectors.shape[0]
202            capacity = fp["vectors"].shape[0]  # pyright: ignore[reportAttributeAccessIssue]
203
204            # check if we have enough space, resize if necessary
205            cur_num_vectors = cast("int", fp.attrs["num_vectors"])
206            space_left = capacity - cur_num_vectors
207            if num_new_vecs > space_left:
208                new_size = max(
209                    cur_num_vectors + num_new_vecs,
210                    capacity + self._resize_min_val,
211                )
212                LOGGER.debug("resizing index from %s to %s", capacity, new_size)
213                for ds in ("vectors", "doc_ids", "psg_ids"):
214                    fp[ds].resize(new_size, axis=0)  # pyright: ignore[reportAttributeAccessIssue]
215
216            # add new document IDs to index and in-memory mappings
217            doc_id_idxs, non_null_doc_ids = [], []
218            for i, doc_id in enumerate(doc_ids):
219                if doc_id is not None:
220                    self._doc_id_to_idx[doc_id].append(cur_num_vectors + i)
221                    doc_id_idxs.append(cur_num_vectors + i)
222                    non_null_doc_ids.append(doc_id)
223            fp["doc_ids"][doc_id_idxs] = non_null_doc_ids  # pyright: ignore[reportIndexIssue]
224
225            # add new passage IDs to index and in-memory mappings
226            psg_id_idxs, non_null_psg_ids = [], []
227            for i, psg_id in enumerate(psg_ids):
228                if psg_id is not None:
229                    self._psg_id_to_idx[psg_id] = cur_num_vectors + i
230                    psg_id_idxs.append(cur_num_vectors + i)
231                    non_null_psg_ids.append(psg_id)
232            fp["psg_ids"][psg_id_idxs] = non_null_psg_ids  # pyright: ignore[reportIndexIssue]
233
234            # add new vectors
235            fp["vectors"][cur_num_vectors : cur_num_vectors + num_new_vecs] = vectors  # pyright: ignore[reportIndexIssue]
236            fp.attrs["num_vectors"] += num_new_vecs  # pyright: ignore[reportOperatorIssue]

244    def _get_vectors(self, ids: "Iterable[str]") -> tuple[np.ndarray, list[str]]:
245        idx_pairs = []
246        for id in ids:
247            if self.mode in (Mode.MAXP, Mode.AVEP) and id in self._doc_id_to_idx:
248                idxs = self._doc_id_to_idx[id]
249            elif self.mode == Mode.FIRSTP and id in self._doc_id_to_idx:
250                idxs = [self._doc_id_to_idx[id][0]]
251            elif self.mode == Mode.PASSAGE and id in self._psg_id_to_idx:
252                idxs = [self._psg_id_to_idx[id]]
253            else:
254                LOGGER.warning("no vectors for %s", id)
255                idxs = []
256
257            for idx in idxs:
258                idx_pairs.append((id, idx))
259
260        # h5py requires accessing the dataset with sorted indices
261        out_ids, vec_idxs = zip(*sorted(idx_pairs, key=lambda x: x[1]))
262        vec_idxs = list(vec_idxs)
263
264        with h5py.File(self._index_file, "r") as fp:
265            # reading all vectors at once slows h5py down significantly, so we read them
266            # in chunks and concatenate
267            vectors = np.concatenate(  # pyright: ignore[reportCallIssue]
268                [
269                    fp["vectors"][  # pyright: ignore[reportIndexIssue, reportArgumentType]
270                        vec_idxs[i : i + self._max_indexing_size]
271                    ]
272                    for i in range(0, len(vec_idxs), self._max_indexing_size)
273                ]
274            )
275            return vectors, list(out_ids)

 55    def _get_num_vectors(self) -> int:
 56        # account for the fact that the first shard might be larger
 57        if len(self._shards) < 2:
 58            return self._idx_in_cur_shard
 59        return (
 60            self._shards[0].shape[0]
 61            + (len(self._shards) - 2) * self._alloc_size
 62            + self._idx_in_cur_shard
 63        )
 64
 65    def _get_internal_dim(self) -> int | None:
 66        if len(self._shards) > 0:
 67            return self._shards[0].shape[-1]
 68        return None

 70    def _add(
 71        self,
 72        vectors: np.ndarray,
 73        doc_ids: IDSequence,
 74        psg_ids: IDSequence,
 75    ) -> None:
 76        # if this is the first call to _add, no shards exist
 77        if len(self._shards) == 0:
 78            self._shards.append(
 79                np.zeros((self._init_size, vectors.shape[-1]), dtype=vectors.dtype)
 80            )
 81
 82        # assign passage and document IDs
 83        for i, doc_id in enumerate(doc_ids, len(self)):
 84            if doc_id is not None:
 85                self._doc_id_to_idx[doc_id].append(i)
 86
 87        for i, psg_id in enumerate(psg_ids, len(self)):
 88            if psg_id is None:
 89                continue
 90            if psg_id in self._psg_id_to_idx:
 91                raise RuntimeError(f"Passage ID {psg_id} already exists.")
 92            self._psg_id_to_idx[psg_id] = i
 93
 94        # add vectors to shards
 95        added = 0
 96        num_vectors = vectors.shape[0]
 97        while added < num_vectors:
 98            cur_shard_size, dim = self._shards[-1].shape
 99
100            # if current shard is full, add a new one
101            if self._idx_in_cur_shard == cur_shard_size:
102                LOGGER.debug("adding new shard")
103                self._shards.append(np.zeros((self._alloc_size, dim)))
104                self._idx_in_cur_shard = 0
105                cur_shard_size = self._alloc_size
106
107            to_add = min(
108                num_vectors - added,
109                cur_shard_size,
110                cur_shard_size - self._idx_in_cur_shard,
111            )
112            self._shards[-1][
113                self._idx_in_cur_shard : self._idx_in_cur_shard + to_add
114            ] = vectors[added : added + to_add]
115            added += to_add
116            self._idx_in_cur_shard += to_add
117
118    def consolidate(self) -> None:
119        """Combine all shards of the index in one contiguous section in the memory."""
120        # combine all shards up to the last one entirely, and take only whats in use of
121        # the last one
122        self._shards = [
123            np.concatenate(
124                self._shards[:-1] + [self._shards[-1][: self._idx_in_cur_shard]]
125            )
126        ]
127        self._idx_in_cur_shard = self._shards[0].shape[0]

133        return set(self._psg_id_to_idx.keys())

135    def _index(self, idxs: "Sequence[int]") -> np.ndarray:
136        """Return vectors from the internal array(s).
137
138        This method retrieves vectors from the correct shards (if there are any).
139
140        :param idxs: Indices to return vectors for.
141        :return: The vectors in the same order as the provided indices.
142        """
143        if len(idxs) == 0:
144            return np.array([])
145
146        # if there is no sharding, simply use numpy indexing
147        if len(self._shards) == 1:
148            return self._shards[0][idxs]
149
150        # otherwise, group indexes by shard and index each shared individually
151        items_by_shard = defaultdict(lambda: ([], []))
152        for i, idx in enumerate(idxs):
153            # the first shard might be larger
154            if idx < self._shards[0].shape[0]:
155                shard_idx = 0
156                idx_in_shard = idx
157            else:
158                idx_ = idx - self._shards[0].shape[0]
159                shard_idx = int(idx_ / self._alloc_size) + 1
160                idx_in_shard = idx_ % self._alloc_size
161            items_by_shard[shard_idx][0].append(idx_in_shard)
162            items_by_shard[shard_idx][1].append(i)
163
164        result = []
165        ordering = []
166        for shard_idx, (idxs_in_shard, i_) in items_by_shard.items():
167            result.append(self._shards[shard_idx][idxs_in_shard])
168            ordering.extend(i_)
169
170        return np.concatenate(result)[np.argsort(ordering)]
171
172    def _get_vectors(self, ids: "Iterable[str]") -> tuple[np.ndarray, list[str]]:
173        idxs = []
174        ids_ = []
175        for id in ids:
176            if self.mode in (Mode.MAXP, Mode.AVEP) and id in self._doc_id_to_idx:
177                idxs.extend(self._doc_id_to_idx[id])
178                ids_.extend([id] * len(self._doc_id_to_idx[id]))
179            elif self.mode == Mode.FIRSTP and id in self._doc_id_to_idx:
180                idxs.append(self._doc_id_to_idx[id][0])
181                ids_.append(id)
182            elif self.mode == Mode.PASSAGE and id in self._psg_id_to_idx:
183                idxs.append(self._psg_id_to_idx[id])
184                ids_.append(id)
185            else:
186                LOGGER.warning("no vectors for %s", id)
187        return self._index(idxs), ids_

190        self, batch_size: int

192        LOGGER.info("creating ID mappings for this index")
193        idx_to_doc_id = {
194            idx: doc_id
195            for doc_id, idxs in tqdm(self._doc_id_to_idx.items())
196            for idx in idxs
197        }
198        idx_to_psg_id = {
199            idx: psg_id for psg_id, idx in tqdm(self._psg_id_to_idx.items())
200        }
201
202        num_vectors = len(self)
203        for i in range(0, num_vectors, batch_size):
204            idxs = range(i, min(i + batch_size, num_vectors))
205            yield (
206                self._index(idxs),
207                list(map(idx_to_doc_id.get, idxs)),
208                list(map(idx_to_psg_id.get, idxs)),
209            )
